# The Flux — Primordial QGP Cosmology

A cosmological model in which the Big Bang is an ongoing process.
Space itself is identified with the primordial quark-gluon plasma (QGP).

## Main result

EOS β formula tested on 163 SPARC galaxies (Lelli et al. 2016):
- Median reduced χ² = **0.57** (vs NFW 1.47, Burkert 0.79, MOND ~1.2–1.4)
- Same number of free parameters (2)
- Multi-scale validation: solar system, galaxies, galaxy clusters
- GR corrections quantified and shown negligible at galactic scales

## Usage

Download SPARC data: https://astroweb.cwru.edu/SPARC/  
Place files in `./sparc_data/` then:
```bash
python3 flux_complet.py
```

Requires: numpy, scipy, matplotlib
